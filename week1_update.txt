DSSE Assignment 1 Week 1 update
Group 6 - Working on apache tika detect and parser

Compiled the tika repository into a .jar file which was analyzed using arcade JavaParser to create the .rsf file.
This .rsf contained all the class dependencies for the tika repository, so we then used filtering.py to filter out the dependencies related to detect and parser only.
After the filtered .rsf was generated, used clustering algorithms (WCA, Limbo and ACDC) with the clustering.py to detect the structural clusters present in the architecture.
WCA and Limbo were run with k=4,9,15 to compare various results. We used UEMNM for WCA and the non-matching algorithm gives a similarity score of 0 providing more accurate results
Once the clustering was completed, we analyzed the clusters using analyze_cluster.py. The results were stored in cluster_results.txt.
